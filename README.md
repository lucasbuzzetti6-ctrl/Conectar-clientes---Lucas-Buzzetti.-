Introdução: 

o que é Git: Basicamente o Git é um sistema de controle de versãoe também serve para salvar diferentes versões de um projeto, acompanhar mudanças. 
o que é GitHub: É uma plataforma online onde você pode armazenar seus repositórios Git na nuvem, Ele permite: guardar seus projetos, visualizar o histórico...
o que é um repositório: É como uma pasta especial onde seu projeto e todo o histórico de versões são guardados. 
o que é o arquivo .gitignore e por que alguns arquivos não devem ser versionados: É um arquivo onde você lista pastas e arquivos que não devem ser enviados para o Git.
o papel do README.md em um projeto: Ele explica algos sobre o seu projeto, como a introdução.
